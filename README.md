# A demo with github api V4 and react-relay

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

## Necessary 1

    yarn install
    yarn run relay

## Necessary 2

    generate `.env.local`
    
    Configuration ENV

## Dev

    yarn start
  
## Production

    yarn run build
  
## test

   yarn run test


## ENV_KEY
    
    GITHUB_API_TOKEN
