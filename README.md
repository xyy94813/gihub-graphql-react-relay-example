# A demo with github api V4 and react-relay

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

## Necessary 1

    yarn install
    yarn run relay
    yarn run injectToken # Enter Git Hub API Token

## Dev

    yarn start
  
## Production

    yarn run build
  
## test

   yarn run test
